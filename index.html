MaNGOSBot

by ike3

This is a modification of MaNGOS (Zero, One, R2, Trinity) server which brings a number of bots online and available for any player. Also allows you to use your account/guild characters as bots.

This mod could be used to play WoW in a single player mode but experiencing multiplayer content.

Core Features

Play with your alts

they are always online with you

Whenever you log in, all your alt characters are online as well. You could interact with them, exchange items, invite to your group, go to a dungeon or raid, check their stats, use their tradeskills, craft items etc.

Populated world

hundreds of bots playing the game

Even if you are the only player on the server, there are random bots available across all the continents. You can meet them in world, invite to your group, trade, go for a dungeon or raid, pvp, duel etc.

Auction house and trading

with bots

Bots will post items on the AH so you could buy something useful. They will also buy your items if the price is right. There is a number of pricing strategies and the market simulation making the prices fluctuate depending on your actions.

Guild tasks

check your mail

Bots will form random guilds sending you advertisement mails for some tasks. If you kill certain mobs or bring the sender certain items (using trade or sending by mail) guild masters will reward you with gold and good items. But make haste though, the tasks expire one a coouple days.

Bot Features

Simplicity

just install MaNGOSBot addon

The best way to interact with bots is to use the MaNGOSBot addon. Just select your bot and click the appropriate buttons. There are keybinding available such as commands to follow you, stop moving, attack etc. You do not need to do anything though - just invite bots to your group, command them to follow and the AI will take care for everything else.

Fine bot control

via chat commands or using MaNGOSBot addon

Bots are self-capable of doing usual WoW stuff, but the power of the mod is the chat commands you could use to tell bots what and how to to it. For example, you can tell them to start clearing the area of mobs while you doing something more interesting. Or command to go to a mailbox, check the mail and maybe send you some items or gold.

AI

combat and non-combat mode

The engine uses reactive AI so the bots watch for something happens and react by doing something. There are different ways how to react so calling strategies, and you can tell bots which one to use at the moment. For example, you can swith a warrior to be a tank or dps. A mage can use frost, fire or arcane rotations. A priest could a healer or a shadow dps. You can tell the bots how to buff players (increasing their dps, mana regen or maybe threat).

Immersive

Buying attrbites

from vendors

You need not only buy your abilities from class trainers but the attributes are purchaseable now. Each level you gain 3 attribute points which can be spent on Strength, Agility, Intellect, Stamina or Spirit. You need a lot of gold for this so playing on the AH with bots is a must.

Death is painful

you lose attrbites

If you died and releases your spirit you lose 3 attribute points and need to buy them again from class trainer. So make sure you do not die. Bring a lot of healing potions, use items etc. You do not lose anything if resurrected by a healer thought.

Fall damage

can kill you

Falling is painful now. Jumping from a height more than twice the character size is a certain death. So you'd better use spells or items to avail falling.

Simlified alt leveling

and reputation grinding

All your alts are leveled with you if they are not in a group. So when you get some XP or reputation they get 90% of the amount gained so you do not need to level 10 your chars independenlty to form a raid group. Completing a quest also marks it as completed for all your alts. They do not receive any rewards and gold thought and you need gold to buy them skills and attributes.

Installation Guide

Installing mangosbot can be tricky but easy if you are famillar with the MaNGOS core itself. Basically you need to enable some settings during compilation and apply additional SQL files to your DB and add config files to your realm.

Workflow
  • Checkout the code from the appropriate branch depending on the game version you want to use. Theare are a little changes to the core itself, mostly some hook points to call the MaNGOSBot code. You can even checkout other version of the core and apply these changes manually (search of #ifdef ENABLE_PLAYERBOTS for more information).
  • Update all submodules to the latest version. The references to submodules are usually outdated so it is a good idea to pull them manually so the latest version will be used. It is always compatible with the main code.
  • Configure and build the code. Make sure to set ENABLE_PLAYERBOTS and ENABLE_IMMERSIVE to true.
  • Apply SQL files from modules/Bots/sql and modules/Immersive/sql to the appropriate DBs (see file names such as characters_*.sql and world_*.sql)
  • Configure AhBot, Immersive and MaNGOSBot by putting ahbot.conf, immersive.conf and aiplayerbot.conf to your config directory (where mangosd.conf is located). Feel free to change any value, although the default ones are fine too. This actually enables the features so the config files are mandatory.
  • Run the server. The first run can be quite slow as a vast number of random characters will be created. It is recommended to init them all at once by running "rndbot init" console command, wait for it to complete and restart the server. If you do not run the server for more than couple days it is a good idea to repeat the initialization process again.

Source Code only

MaNGOSBot is available as source code only. You need to build it using the appropriate tools for your platform.

How to play the game

Create your main character and a number of alts. As they will level simultaniously it is a good idea to create them early on. Log in the game as a main character.

You'll notice than all your alts are staying near you ready to play. Invite them to your group and use MaNGOSBot Addon to customize their behaviour.

It is recommended to assign some keyboard shortcuts to MaNGOSBot Addon key binding such as Follow, Stay, Attack, Flee etc to quickly command all your party to do something. Now make them follow you and go play the game!

Grab a quest from a quest giver - your alts will grab it too. Start killing the mobs for the quest - your alts will help you: the tank will grab the aggro, the healer will heal, others will damage your targets, everyone will apply the needed buffs etc. They will loot the quest items if necessary and report to chat their progress with quests.

If the quest requires to use some gameobjects you can ask bots to tell you all they see. Whisper:

los

This will print you a lot of links: all game object, units, npcs, players etc. nearby. Choose the correct one and tell: 

u [gameobject]
[gameobject] is the gameobject link, Shift-click the links provided by bot and it will be copied to the chat box. The bot willthen use the object and trigger the quest required action.

When the quest is done for everyone in your group, go to the quest giver and turn it in. The bots will turn the quest too and get the XP and the reward if available.

If the quest has multiple rewards, you need to tell everyone what to choose. Just whisper to each player a command such as: 

r [item]
[item] is a item link, Shift-click to the quest reward to copy it to the chat box. Then the bot will turn the quest in choosing this reward.

When looting you'll notice that bots sometimes say to chat that the looted items could be useful in some way. They could equip it for instance. Or you may have an item useful for a bot or maybe they will have an item useful for you. It is time to exchange those items and equip something new. Just select a bot and click Inventory button on the MaNGOSBot Addon. Provided you installed EngBot addon the window will pop-up showing the bot's inventory the same way this addons shows yours. Use context menu to make the bot add the item to the trade window, to equip, to use, or even to destroy the items. You can give them your items and gold by trading.

Alternative to the addon are the chat commands. You can find more information in latter pages on this guide, but in general you need to whisper commands such as (small text is the command description): 

u [item] to use this item
e [item] to equip this item
ue [item] to unequip this item
t [item] to add the item to the trade window
b [item] to buy this item from a vendor nearby
s [item] to sell this item
destroy [item] to destroy the item

Now it is time to level up some skills and attributes. Go to the class trainer and start talking. The bots will whisper in chat what they could learn. Tell them: 

trainer learn
And the bot will learn these skills.

Buying attributes is done by talking to the npc. This is done using the talk command:

talk

And the bot will whisper you everything the NPC said to him. Note than options will have a numbers before them. Whisper:

talk 2
2 - is the number of an option and the bot will choose this option and the conversation will go on. So you can buy the attributes now.

You can learn this way any skills including weapons, crafting, spells etc.

Hey, they have not enough gold to do so! Is it time to play the game and get some stuff to sell!

Now it is time to craft something. Use MaNGOSBot Addon and click Spells button. The window will pop-up showing all craftable items that bot can craft. Just use context menu to craft it. They could create something useful for you too!

Sometimes you need to check some information about the bot current stats, level etc. There are some commands available: 

stats to see bot gold, XP, item durability
who to see bot overall information such as level, talents etc
who [skill] to see level of that skill, eg. skinning
ready to see overall bot readyness to fight

Bored to do a group play? Then disband the party and play alone for some time. Is it usually a good idea to brind your alts near a mailbox or to make them stay at the inn (maybe using a hearthstone) before though. Do not worry: the bots will level up with you, so you won't outlevel them.

The easiest way to disband the party is to whisper: 

leave
in the group chat or use the MaNGOSBot addon. Note that if you leave the party, others will still remain in party without you and they will no t get any shared XP, rep, do quests etc. so this commands makes all bots to leave the party disbanding it.

OK, now it is time to solo!

Got something useful for others during solo play? Send it to your alts via a mailbox. Even if you are far away from them, whisper: 

mail ?
Provided your alts were near a mailbox, they will show all their mail in chat. Again, note numbers before mails - they are used in mail commands: 
mail take 5 - Take all items from mail number 5
mail read 5 - Read mail number 5
mail delete 5 - Delete mail number 5
sendmail [item] - Send [item] to you
sendmail [item] [player] - Send [item] to a specific [player]

Did some leveling? Your alts needs it too! Summon them to use or ask them to summon you: 

summon
Summoning player need to be near a meeting stone or an innkeeper to be able to do so. That's why you brought your alts to the inn!

Maybe now it is time to go for a dungeon? Do not forget to grab the necessary quests or just share yours with bots.

Happy playing!
